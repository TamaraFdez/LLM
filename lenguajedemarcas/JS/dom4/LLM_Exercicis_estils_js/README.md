# Pràctica: tocar estils amb JavaScript (DOM)

## Normes (IMPORTANT)
- **Només pots modificar `script.js`.**
- **Només pots afegir codi on posa `// TODO`.**
- No pots canviar res més del fitxer: no renombrar variables, no afegir funcions fora dels TODO, no modificar comentaris.
- **NO es pot modificar `index.html` ni `styles.css`.**
- **Entrega només `script.js` sencer** (el mateix fitxer, amb els TODO completats).
